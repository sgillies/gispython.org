The GIS and Python Software Laboratory

This site was begun in 2005 to provide infrastructure for developing open source GIS tools for Python programmers. Times change and infrastructure does too. The OWSLib project has moved to Sourceforge: http://sourceforge.net/apps/trac/owslib/wiki. The Zope projects are being superceded by the collective.geo suite at CoActivate: http://www.coactivate.org/projects/collectivegeo/project-home. Everything else (Fiona, Rtree, and Shapely) has been moved to a GitHub organization: http://github.com/Toblerity.

Help and discussion about the projects continues on our email list (archives).
